rf2livetiming

  • node.js web application
  • uses socket.io to deliver updates in real time
  • serves current session information for a single rF2 server
  • personal bests are highlighted green, class bests are highlighted purple
  • AI drivers shown in a grayer font color than humans
  • tracks best laps for each driver/car/track combination
  • generates a track map based on a driver's path around the track
  • each class is shown in a different color, which can be customized
  • displays said track and current vehicle locations with class color coding in a canvas
  • laps and maps stored in json

rf2livetiming-plugin

  • c++ rFactor 2 plugin
  • sends scoring updates over UDP
  • reads and sends the lastest results xml file over TCP
  • ideally networking over loopback only but can send data elsewhere
